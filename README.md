### 📦 Dependencies | Dependências

```bash
python: >= 3.7
textblob: >= 0.15.3
```

### 🚀 Getting Started | Começando

```bash
# Clone the repository | Clone o repositório
git clone https://github.com/matheussricardoo/story-generator.git

# Navigate to project directory | Navegue até o diretório do projeto
cd story-generator

# Install dependencies | Instale as dependências
pip install textblob
python -m textblob.download_corpora

# Run the generator | Execute o gerador
python gerador_historias.py
```
